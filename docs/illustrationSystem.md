# Illustration System
## Sammy and the Brainrot Paradox

This document ensures visual consistency for all generated illustrations.

It should be used as the base prompt when generating new artwork.

---

# Core Visual Style

Illustrations should resemble the Roblox game aesthetic.

Characteristics:

- block-style Roblox characters
- bright glowing objects
- simple geometric environments
- clean lighting
- stylized digital atmosphere

Avoid painterly illustration styles.

The goal is to visually resemble a **Roblox gameplay world**.

---

# Character Design

- casual clothing

Environment:

Sammy often appears in a real-world room with a computer setup.

# Brainrot Visual Design

Brainrots should appear as floating glowing objects.

They should resemble collectible game artifacts.

---

## Speed Brainrot

Blue glowing orb with lightning streaks.

---

## Echo Brainrot

Purple orb with repeating circular rings.

---

## Lag Brainrot

Green cube with glitch-like flickering edges.

---

## Golden Brainrot

Gold sphere with orbiting rings.

---

## Founder Brainrot

Floating silver crown made of shifting geometric shapes.

It should appear powerful and mysterious.

---

## Observer Brainrot

Soft white orb with calm glow patterns.

Represents understanding rather than power.

---

# Environment Style

Game world environments should look like stylized Roblox maps.

Examples:

- floating platforms
- block architecture
- glowing skies
- minimal textures
- bright lighting

---

# Composition Guidelines

Each illustration should:

- focus on one key moment from the story
- clearly show the brainrot object involved
- maintain a sense of scale and action

Characters should remain readable and expressive.

---

# Illustration Consistency Rules

Always maintain:

- same brainrot shapes
- same character proportions
- consistent Roblox-style rendering
- bright glowing effects for powers

These rules ensure visual continuity across the book.
